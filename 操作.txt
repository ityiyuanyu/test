git init初始化
git add . 将文件添加到暂存区
git commit -m "commit" 提交到历史版本
git remote add origin 远程仓库地址 //本地添加远程仓库地址
git push -u origin master 推送master到指定远程地址

新建文件
git add . 将文件添加到暂存区
git commit -m "commit" 提交到历史版本
直接git push

拉取远程仓库最新数据
git pull

合并分支后删除远程仓库分支 git push origin --delete 要删除的分支
git branch -d 要删除的分支
